# Executive-Sales-Dashboard-PowerBI
Interactive Executive Sales Dashboard built using Power BI.

## Project Overview

This is an interactive Power BI dashboard created to analyze sales performance.

## Tools Used

- Power BI
- Excel
- DAX

## Dashboard Features

- Monthly Profit Trend
- Revenue by State
- Profit by Product Sub-Category
- Sales Quantity by Category
- Payment Method Distribution
- Top Customers by Sales

## Key Insights

- Maharashtra generated the highest sales.
- Clothing had the highest sales quantity.
- COD was the most preferred payment method.
- Printers generated the highest profit.
